# Déploiement Netlify/Vercel

- Netlify : connecte ton repo GitHub et déploie. Build command : `npm run build` ; Publish directory : `.next`
- Vercel : connecte ton repo GitHub et déploie. Build command : `npm run build` ; Output : `.next`
- Pour un export statique (GitHub Pages) :
    npm run build
    npm run export
    (le dossier `out/` sera généré)

Le sitemap.xml est déjà présent.
La page "À propos" est accessible sur /about.
